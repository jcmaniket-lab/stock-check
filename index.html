🔗
Connect App → Google Sheets
One-time setup · Takes 5 minutes · Free forever
1
Create a new Google Sheet

Go to sheets.google.com → click Blank to create a new spreadsheet.

Name it "Stock Check" at the top.

2
Open Apps Script

In the sheet menu click: Extensions → Apps Script

A new tab opens with the code editor. Delete everything in it.

3
Paste the WebApp.gs code

Open the file 📄 WebApp.gs you downloaded. Copy all of its contents and paste it into the Apps Script editor.

Click the 💾 Save icon. Name the project Stock Check.

4
Deploy as a Web App

Click the blue Deploy button (top right) → New deployment.

Click the gear ⚙️ next to "Select type" → choose Web app.

Set these options exactly:

Execute as: → Me
Who has access: → Anyone

Click Deploy. Google will ask for permissions — click Authorize access → Allow.

⚠️ You must choose "Anyone" — otherwise the phone app can't send data to it.
5
Copy your Web App URL

After deploying, Google shows you a Web App URL. It looks like this:

https://script.google.com/macros/s/AKfycby.../exec

Copy this URL — this is what connects your phone app to the sheet.

✅ You can test it by opening the URL in a browser — you should see: "Stock Check Web App is running ✅"
6
Paste the URL into the Stock Check app

Open your Stock Check web app on your phone (or on GitHub Pages).

On the start screen, paste the Web App URL into the "Google Sheets URL" field.

The app remembers it — employees only need to paste it once per device.

ℹ️ When they finish a check and tap "Save & Finish", all rows save automatically to your Google Sheet in the exact format shown in your screenshot.
7
Share the sheet with your team (optional)

In Google Sheets, click Share (top right) and add anyone who needs to view the results — managers, supervisors, etc.

Give them Viewer access so they can see data but not accidentally edit it.

💡 The sheet updates live — as soon as an employee finishes a check on their phone, the rows appear in the sheet instantly.
